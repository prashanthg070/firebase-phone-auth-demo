How It Works

Our application leverages Firebase Phone Number Verification to authenticate users:

  1. User launches the app and sees the verification screen
  2. App checks if carrier supports instant verification
  3. If supported, user grants permission for instant verification
  4. Phone number is verified securely via carrier network
  5. If not supported, SMS verification is used as fallback
